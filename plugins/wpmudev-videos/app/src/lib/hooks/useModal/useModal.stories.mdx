# useModal

Opens an SUI modal

## Usage

### Parameters

The hook accepts a single parameter of type object with the following parameters. <br />

| key            | type       | description                                                 |
| -------------- | ---------- | ----------------------------------------------------------- |
| id             | `string`   | unique modal container id                                   |
| hasOverlayMask | `boolean`  | Whether to add an overlay or not                            |
| onOpen         | `function` | callback function to be executed when the modal gets opened |
| onClose        | `function` | callback function to be executed when the modal gets closed |

### Return

Since it uses Array destructuring, you can use custom names, the first parameter is a functio to open the modal, the second parameter is a function to close it,

| function   | description                      |
| ---------- | -------------------------------- |
| openModal  | opens the modal                  |
| closeModal | closes the modal                 |
| isOpen     | whether the modal is open or not |

-   Modal content needs to be wrrapped with [the Modal component](/docs/lib-components-modal--page)

### Example

```
import { useModal } from '{/}'
import { Modal, Box } from '{/}'

function Example() {
	const modalID = 'wpmudev-videos-data-reset-confirmation'

	const [ openModal, closeModal ] = useModal({
		id: modalID
	})

	return (
		<>
			{/* Button to open the modal */}
			<button id={`${modalID}-opener`} onClick={openModal}>
				Open Modal
			</button>

			<Modal id={modalID}>
				<Box>
					<Box.Header className="sui-flatten sui-content-center sui-spacing-top--60">
						<button
							className="sui-button-icon sui-button-float--right"
							onClick={closeModal}
							id={`${modalID}-close`}
						>
							<span
								className="sui-icon-close sui-md"
								aria-hidden="true"
							></span>
							<span className="sui-screen-reader-text"></span>
						</button>

						<h3
							id={`${modalID}-title`}
							className="sui-box-title sui-lg"
						>
							Modal Title
						</h3>

						<p id={`${modalID}-desc`} className="sui-description">
							Modal Description
						</p>
					</Box.Header>
					<Box.Body>Content in the Body</Box.Body>
					<Box.Footer>Content in the Footer</Box.Footer>
				</Box>
			</Modal>
		</>
	)
}

export default Example

```
