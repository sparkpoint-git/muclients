# api

API related helper functions for admin.

## Available functions

All available functions use `apiFetch` function from [@wordpress/api-fetch](https://www.npmjs.com/package/@wordpress/api-fetch?activeTab=versions) package with nonce middleware

### restGet

Sends an API rest GET request

### restPost

Sends an API rest POST request

### restDelete

Sends API rest DELETE request

refer to `api.js` for usage details
